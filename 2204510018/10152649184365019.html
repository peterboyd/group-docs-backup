Recommended distro to start :


Xubuntu 12.04.4 LTS


32 bits or 64 bits :


Under 4Gb of ram 32 bits is better. At 4Gb or better, use 64 bits or a 32 bits PAE kernel that will allow to use the ram over 3,5Gb


Easy suggestion to make it even more lightweight and fast :


sudo apt-get install openbox obconf
