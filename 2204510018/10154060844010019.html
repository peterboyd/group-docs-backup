TITLE OF THE DOCUMENT (just copy/paste it):
Things to do after installing Fedora 20 (Assuming the installation is done from the 4.3 GB Fedora 20, i386 Installation ISO.)
CONTENT OF THE DOCUMENT (just copy/paste it):

1. Install "Russian Fedora" Reposotories (Needed for VLC, Rosa Media Player etc):


sudo yum localinstall http://mirror.yandex.ru/fedora/russianfedora/russianfedora/free/fedora/releases/20/Everything/i386/os/russianfedora-free-release-20-1.R.noarch.rpm


sudo yum localinstall http://mirror.yandex.ru/fedora/russianfedora/russianfedora/nonfree/fedora/releases/20/Everything/i386/os/russianfedora-nonfree-release-20-1.R.noarch.rpm

2. Install "RPM Fusion" Repositories (Needed for Unrar, Xine, Audacity etc):


su -c 'yum localinstall --nogpgcheck http://download1.rpmfusion.org/free/fedora/rpmfusion-free-release-$(rpm -E %fedora).noarch.rpm http://download1.rpmfusion.org/nonfree/fedora/rpmfusion-nonfree-release-$(rpm -E %fedora).noarch.rpm'

3. Install everything from the 4.3 GB Fedora 20 installation ISO:


Step 01: Create a empty directory 'F20_ISO' in '/mnt' ($ sudo mkdir /mnt/F20_ISO),
   needed for mounting the ISO.

Step 02: Assuming the ISO is located in - /home/soham/BackUps/ISOs, mount the ISO:
   $ sudo mount /home/soham/BackUps/ISOs/Fedora-20-i386-DVD.iso /mnt/F20_ISO/

Step 03: Now we need to create a fake repo 'fedora-dvd.repo' in /etc/yum.repos.d with the followings:
   $ sudo leafpad /etc/yum.repos.d/fedora-dvd.repo
   Copy/Paste the following:

[fedora-dvd]
name=Fedora DVD $releasever – $basearch
failovermethod=priority
baseurl=file:///mnt/F20_ISO
enabled=1
metadata_expire=7d
gpgcheck=1
gpgkey=file:///etc/pki/rpm-gpg/RPM-GPG-KEY-fedora-$basearch

Save the file and exit Leafpad.

Step 04: Now press 'ALT+F2', type the command: gpk-prefs (This will bring the 'Software Update Prefrences' dialog.)
   Go to the "Software Sources" tab and uncheck everything except 'Fedora DVD 20 -i386'.
   Click OK and exit the dialog.

Step 05: To see what is installable inside the DVD, issue:
   $ sudo yum grouplist

   This will bringup a list of group packages which are either available or already installed.

   To install something like the Sugar Desktop Environment, enter the command like this:

   $ sudo yum groupinstall "Sugar Desktop Environment"

   Just replace "Sugar Desktop Environment" with any other string listed by 'grouplist' command to install that.

Step 06: After finishing installing from that group list, to see what else available within the ISO,
   we need to install "Yumex" - A GUI for YUM.

   $ sudo yum install yumex
  
   Note: You will have to enable other repos and the Internet for this download/install to work.
   Follow 'Step 04', Check everything else, except ''Fedora DVD 20 -i386' repo.

   To open YUMEX press 'ALT+F2', and then type: yumex
  
   Go to 'Active Reposotories' tab of Yumex and then check only the "fedora-dvd" repo
   After that click on the 'Refresh' button to load the list of packages.

   Now click the 'Available' radio button of Yumex to filter out only the installable packages left in the ISO.
   Dont use the search box, rather manually look for the packages to install and mark them.

   After you finish , click on Install. Rest follow the on-screen dialogs.

Step 07: After finishing everything, delete the 'fedora-dvd.repo':
   $ sudo rm /etc/yum.repos.d/fedora-dvd.repo
   Re-enable all other repos by following the 'Step 04', by doing the reverse of what you did in 'Step 04'.

Step 08: Delete the mount point.
  
   $ sudo rmdir /mnt/F20_ISO
  
Thats all.
