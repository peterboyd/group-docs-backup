Installing software on openSuse using Yast

The process of installing software is very simple. Start YaST by selecting it from the menu under System, or by using the run command dialog (press Alt+F2) and typing yast. You will be required to enter your root password. Start the Software Management module by selecting it from the Software tab in the YaST Control Center.

Search for the required package, and YaST will display a list of all matching packages on the right. You can search for package names or you can search in package descriptions. There are also filters available that enable you to browse certain categories of packages.

Select the packages that you wish to install and click 'Accept'. If some additional packages are required for the selected packages to be installed, YaST will prompt for confirmation. Once confirmed, YaST will fetch the packages from installation repositories and install them.

 

 

Software availability

The number of packages available in YaST Software Management depends on the repositories added. By default you will only have the CDs/DVD or the official online repositories that you can add during installation. To add additional repositories:

  • Start YaST by clicking on it under Software in your menu or by typing yast in the run command box (press Alt+F2).
  • Select Software Repositories.
  • Click on Add, select Specify URL.
  •  Then enter a Repository Name and a URL. It's recommended to copy/paste URLs to avoid typos (you can get repository urls here: http://en.opensuse.org/Package_repositories and even more here: http://en.opensuse.org/Additional_package_repositories )
  • And click 'Next'

Package metadata will be downloaded and parsed - this takes time depending on mirror speed, your bandwidth, the size of the repository, the speed of your system.

 

 

NOTES:

  • You can remove or disable repositories at any time
  • In case you have downloaded some RPM files, you can place them in one folder on your pc, then add that folder as a repository in Yast by selecting "local folder" where you'd have selected "specify url" (an easy way of creating your own repository)
  • You can also specify to Yast to keep all downloaded installation files from certain repositories. You can later find these under /var/cache/zypp/packages and you'll be able to copy these to create your own local repository to share with friends, or to use in case you have to reinstall the system.
  • openSuse uses the red-hat package management system(rpm) and you can also use the rpm command to install/upgrade/uninstall software on your PC.
