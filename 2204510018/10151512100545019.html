Package management made easy:

 

packer -Sy package1 package2 ... install packages

sudo pacman -U /packagefolder/package = install package offline

sudo pacman -Syy = sync mirrors after pacman.conf/mirrorlist changespacker -Syu = update your system including Aur

packer keyword = search a package

sudo pacman -Q > list.txt = list all installed packages in the current folder

sudo pacman -R package1 package2 ... = remove packages

sudo pacman -Rns package1 package2 ... = remove packages, dependancies, configuration

yaourt -Sy package1 package2 ... --tmp /home/mike/Desktop/Aur = install bigger packages that may be too big for /tmp, don't forget to empty your /tmp before ...

/etc/makepkg.conf Makepkg configuarion

file/etc/pacman.conf Pacman config

file/etc/abs.conf ABS config

file/etc/mkinitcpio.conf Initrd cofiguration

/etc/conf.d/ Daemons configuration/etc/pacman.d/ Mirrors config for pacman

/etc/rc.d/ Daemons start scripts

 

Xorg configuration

pacman -S xorg Installs xorg

pacman -Ss xf86-video List generic video drivers

Xorg -configurenvidia-xconfig Configure a xorg.conf file [New versions of Xorg should detect everything, install hal with pacman and then try startx] In order to boot in X you should add a login manager to the daemons array (GDM, Slim, Kdm), use the startx script or change the initlevel and login manager in /etc/inittab

Other Common Packages

pacman -S ttf-ms-fonts ttf-dejavu ttf-bitstream-vera Some fonts

pacman -S gnome gnome-extra gdm Installs Gnome, extra apps and the GDM login manager

pacman -S elinks Text mode browser

pacman -S firefox jre flashplugin Installs Firefox with JRE and Adobe Flash plugins

pacman -S vimdiff Useful to check differences betwen files, like a .pacnew config file

pacman -S alsa-lib alsa-utils Installs alsa, then run alsaconfig

pacman -S codecs gstreamer0.10-bad gstreamer0.10-ugly gstreamer0.10-ffmpeg gstreamer0.10-ugly-plugins Propietary codecs
