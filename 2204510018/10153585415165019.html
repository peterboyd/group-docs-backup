2- Code to repair multiples usb devices :

 

cd /sys/bus/pci/drivers/ehci_hcd/

sudo sh -c 'find ./ -name "0000:00:*" -print| sed "s/\.\///">unbind'

 

If useful add it to /ect/rc.local properly ... ;)


With Debian, the fix don't work, this one does ...

/etc/rc.local

rmmod ehci_pci && rmmod ehci_hcd && modprobe uhci_hcd
