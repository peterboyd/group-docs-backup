What to do when Network drivers are missing after installation
CONTENT OF THE DOCUMENT (just copy/paste it):

You bought new laptop, and during and after installation there are no recognized network interfaces. What now?

 

NOTE: I will expand this doc later.

 

1. log in into console as root

2. run "lspci -v" and check for "lscpci -nn"

3. Find what kmod package in ElRepo supports it. link to DeviceID

4. download and install via "yum install <file(s)> --nogpg"

5. Use install DVD (Media repo) to reinstall kernel (so drivers are actually hooked to kernel).

6. run "system-config-network" as root so the /etc/system/network file is created.

6. Reboot
