TITLE OF THE DOCUMENT (just copy/paste it):
Command to show info about hardware device (in this case VGA card) and kernel it uses
CONTENT OF THE DOCUMENT (just copy/paste it):

lspci -nnks $(lspci | grep VGA | awk '{print $1}')


If you change "VGA" part with some other like "Ethernet" you can get info about different device.


Command "lspci" shows all hw devices in the system.


First in row is "ID number" that you can use with "-s" switch to print only about that device.

Switches "-nnk" will show details about all or only selected device including kernel module used.

So "lspci -nnks " will show you details about specific device.
